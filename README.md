# Exercise - Interpreter JS

练习使用 JS 编写简单的解析器。

无第三方库依赖。

## Unit Tests

```bash
$ npm test
```

或者

```bash
$ node test/test.js
```

## 运行脚本

```bash
$ npm run eval script.toy
```

或者

```bash
$ node index.js script.toy
```

其中 `script.toy` 是脚本文件的名称（可以是相对或绝对路径）。

## 打印脚本的 AST

```bash
$ npm run eval script.toy -- --ast
```

或者

```bash
$ node index.js script.toy --ast
```

其中 `script.toy` 是脚本文件的名称（可以是相对或绝对路径）。
